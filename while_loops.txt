3 parts of while / for loops
1. Initialize a control variable
2. Test the control variable
3. Give the control variable a chance to change 

break keyword will leave a for loop

use a while loop when you do not know how many times it needs to be testes, but it is a finiite amount of times

use a do while when you need to run the code at least once before checking the condition

use a for loop when you need to run a block of code a specific amount of times
